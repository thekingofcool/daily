八字的核心是将一个人出生的年月日时转为天干地支，一共八个字。古人认为人出生那一刻的时空环境（金木水火土 五行气场）会影响一个人的性格和体质，从而决定其一生的运势。

构成八字的天干地支，包含：

十天干：甲、乙（木）、丙、丁（火）、戊、己（土）、庚、辛（金）、壬、癸（水）。

十二地支：即生肖序列（子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥）。

这八个字由四柱组成，

年柱的天干地支以立春为界，查万年历得；

月柱地支按节气划分：立春 (寅) / 惊蛰 (卯) / 清明 (辰) / 立夏 (巳) / 芒种 (午) / 小暑 (未) / 立秋 (申) / 白露 (酉) / 寒露 (戌) / 立冬 (亥) / 大雪 (子) / 小寒 (丑)

月柱的天干按年柱的天干推算：

甲己之年丙作首：年干是甲或己，正月就是丙寅月。

乙庚之年戊为头：年干是乙或庚，正月就是戊寅月。

丙辛之年寻庚上：年干是丙或辛，正月就是庚寅月。

丁壬壬位顺行流：年干是丁或壬，正月就是壬寅月。

戊癸之年何方发？甲寅之上好追求：年干是戊或癸，正月就是甲寅月。

日柱每 60 天为一个完整循环，因月大月小及闰年的存在，需查万年历。

时柱的地支是固定的刻度：子时（23:00 - 01:00） / 丑时（01:00 - 03:00） / 寅时（03:00 - 05:00） / 卯时（05:00 - 07:00） / 辰时（07:00 - 09:00） / 巳时（09:00 - 11:00） / 午时（11:00 - 13:00） / 未时（13:00 - 15:00） / 申时（15:00 - 17:00） / 酉时（17:00 - 19:00） / 戌时（19:00 - 21:00） / 亥时（21:00 - 23:00）

时柱的天干是按日柱的天干推算：

甲己还加甲：日干是甲或己，子时就是甲子。

乙庚丙作初：日干是乙或庚，子时就是丙子。

丙辛从戊起：日干是丙或辛，子时就是戊子。

丁壬庚子居：日干是丁或壬，子时就是庚子。

戊癸何方发，壬子是真途：日干是戊或癸，子时就是壬子。

解读八字，首先看日柱的天干，即“日主”：“日主”代表你自己，它是你的基本能量属性。

根据”日主“在出生的月份里是“得令”还是“失令”来判断强弱。

除了“日主”以外的其他七个字与“日主”的关系被看作“十神”，从中看出人生的不同领域的走向。

八字讲究中庸，如果八字中的某个能量太强，即“用神”，就需要“克，泄，耗”它的力量，如果太弱就需要“生，扶”它的力量，当某年的五行刚好是你的“用神”，通常就是你转运顺遂的时候。

八字中，日主是我：

木（甲、乙）：向上生长、仁慈、倔强，火（丙、丁）：热情奔放、礼明、急躁，土（戊、己）：厚德载物、诚实、固执，金（庚、辛）：刚毅果决、义气、冷酷，水（壬、癸）：通达多智、灵动、多疑。

月令（月支）是环境：

得令（强）：月令的五行与日主相同（同类助我），或者月令生日主（长辈扶我）。

失令（弱）：月令克日主，或者日主去生/克月令（消耗体力）。

五行相生：木生火、火生土、土生金、金生水、水生木；相克：木克土、土克水、水克火、火克金、金克木。

八字里的位置关系：

年柱：代表祖上、童年

月柱：代表父母、青年、事业环境

日支：代表内心、配偶

时柱：代表子女、晚年

对于日主为金的十神关系：

我生者（金生水）,水,食神 / 伤官,才华、表达、下属、投资、自由  

我克者（金克木）,木,正财 / 偏财,财富、勤奋、男命的妻子、物质欲  

克我者（火克金）,火,正官 / 七杀,事业、压力、自律、女命的丈夫、法律  

生我者（土生金）,土,正印 / 偏印,学历、长辈、房产、保护、名誉  

同我者（金见金）,金,比肩 / 劫财,朋友、竞争、兄弟、意志力
